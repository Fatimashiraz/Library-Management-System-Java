# Library Management System (Java)

This project is a simple Java console-based Library Management System developed for the Object Oriented Programming course.

## OOP Concepts Used

1. Encapsulation
Private attributes with getters and setters.

2. Inheritance
Book and Magazine classes inherit from LibraryItem.

3. Abstraction
LibraryItem is an abstract class.

4. Polymorphism
Overriding displayInfo() method in subclasses.

## Classes

- LibraryItem (Abstract class)
- Book
- Magazine
- Member
- Main

## Features

- Add books
- Add magazines
- Add members
- Display library items
- Borrow items
